A beautiful template for React Native, Support scaling size.

## Usage

```js
npx react-native init MyApp --template rn-typescript-template
```

## Features

- React 18.1.0</br>
- React Native 0.70.6</br>
- React Navigation V6</br>
- I18Next</br>
